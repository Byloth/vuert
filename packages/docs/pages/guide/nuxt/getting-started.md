# Getting started

## Installation

::: code-group

```bash [NPM]
npm install @byloth/vuert
```

```bash [Yarn]
yarn add @byloth/vuert
```

:::

## Configuration

```ts
// nuxt.config.ts

export default defineNuxtConfig({
  modules: ["@byloth/nuxt-vuert-module"],
  vuert: { /* Optional configuration */ }
});
```
